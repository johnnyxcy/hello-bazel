# Hello Bazel

This is a simple example of a Bazel project with Multi-language support.

1. Node.js
   - TypeScript
   - Electron
   - Vite/Vitest
   - ESM
   - Pnpm Monorepo Setup
2. Python
   - Virtual Environment
   - Pybind11
   - Pytest
   - Poetry Lock
   - Ruff
3. C++
   - LLVM Toolchain
   - GTest
   - Pybind11 + Stubgen
   - Clangd Compile Commands
